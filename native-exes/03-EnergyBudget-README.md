# 打工人能量预算

对应程序：`03-EnergyBudget.exe`

## 用途

把任务从“时间管理”改成“精力管理”，帮助上班族判断今天还剩多少电，以及哪些任务应该降载。

## 适合人群

- 高压白领
- 互联网、金融、咨询、运营、销售等高强度岗位
- 经常下班后完全没电的人

## 核心功能

- 记录起床能量
- 记录关键任务和预计耗能
- 记录压力和睡眠
- 记录恢复动作
- 自动计算剩余电量
- 给出任务排序和降载建议
- 导出 CSV

## 数据位置

`%APPDATA%\TenNativeApps\energy-budget\records.tsv`

## 使用方式

1. 双击 `03-EnergyBudget.exe`
2. 早上填写能量、睡眠和关键任务
3. 保存后查看剩余电量
4. 根据建议安排高耗能任务
5. 晚上补充恢复动作
