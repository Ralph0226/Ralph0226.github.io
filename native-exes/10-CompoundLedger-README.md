# 学习工作复利账本

对应程序：`10-CompoundLedger.exe`

## 用途

记录每天投入在学习、写作、运动、工作和整理上的微小进步，把长期积累变成可见的复利曲线。

## 适合人群

- 本硕博学生
- 考证、转行、学习新技能的人
- 需要长期正反馈的人

## 核心功能

- 创建复利项目
- 标记类别
- 记录今日投入分钟
- 记录今日产出
- 记录质量自评
- 自动生成复利进度评分
- 导出 CSV

## 数据位置

`%APPDATA%\TenNativeApps\compound-ledger\records.tsv`

## 使用方式

1. 双击 `10-CompoundLedger.exe`
2. 记录今天投入了多少分钟
3. 写下产出和质量评分
4. 连续记录，观察长期积累
