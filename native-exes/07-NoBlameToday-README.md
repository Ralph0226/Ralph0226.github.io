# 今日不背锅

对应程序：`07-NoBlameToday.exe`

## 用途

记录关键沟通、口头需求、需求变更和责任边界，生成礼貌确认话术，降低职场背锅风险。

## 适合人群

- 产品、项目、运营、行政、销售、研发
- 经常遇到口头需求和临时变更的人
- 希望留痕但不想显得强硬的人

## 核心功能

- 记录事件标题和相关人
- 记录来源：口头、微信、会议、邮件、文档
- 标记风险等级
- 记录影响范围
- 生成确认话术建议
- 导出 CSV

## 数据位置

`%APPDATA%\TenNativeApps\no-blame-today\records.tsv`

## 使用方式

1. 双击 `07-NoBlameToday.exe`
2. 遇到变更或口头需求时快速记录
3. 保存后查看边界安全度
4. 点击“复制传播文案”或手动复制确认话术发送给对方
