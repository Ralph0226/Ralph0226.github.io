# 会议废话压缩器

对应程序：`05-MeetingCompressor.exe`

## 用途

把会议压缩成结论、负责人、截止日期和待办，减少会后没人执行、责任不清的问题。

## 适合人群

- 项目经理、产品经理、运营、人事
- 经常开会的人
- 需要快速产出会议纪要的人

## 核心功能

- 记录会议主题、参会人数和会议时长
- 估算会议成本
- 记录最终结论
- 记录负责人和截止日期
- 记录待办事项
- 自动生成会议有效度
- 导出 CSV

## 数据位置

`%APPDATA%\TenNativeApps\meeting-compressor\records.tsv`

## 使用方式

1. 双击 `05-MeetingCompressor.exe`
2. 会议中记录结论、负责人和待办
3. 保存后查看会议有效度
4. 会后导出 CSV 或复制传播文案
