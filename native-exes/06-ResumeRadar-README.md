# 简历战斗力雷达

对应程序：`06-ResumeRadar.exe`

## 用途

检查简历和目标岗位的匹配度，帮助学生和职场人优化关键词、量化成果和经历表达。

## 适合人群

- 本科、硕士、博士求职者
- 准备跳槽的白领
- 投递简历但反馈少的人

## 核心功能

- 记录目标岗位
- 记录岗位关键词
- 记录个人技能和核心经历
- 记录量化成果数量
- 记录简历版本数
- 自动生成简历战斗力评分
- 导出 CSV

## 数据位置

`%APPDATA%\TenNativeApps\resume-radar\records.tsv`

## 使用方式

1. 双击 `06-ResumeRadar.exe`
2. 填写目标岗位、关键词和经历
3. 保存后查看战斗力评分
4. 根据建议补关键词和量化成果
