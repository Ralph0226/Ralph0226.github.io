# 电脑工伤账本

对应程序：`01-ComputerInjuryLedger.exe`

## 用途

记录电脑工作者每天的屏幕工作时长、连续久坐、眼疲劳、肩颈痛和休息次数，生成健康损耗评分和恢复建议。

## 适合人群

- 白领、程序员、设计师、研究生、博士生
- 长时间面对电脑的人
- 经常眼干、肩颈痛、腰背酸的人

## 核心功能

- 记录屏幕工作小时
- 记录最长连续久坐小时
- 记录眼疲劳和肩颈痛评分
- 记录有效休息次数
- 自动生成健康余量评分
- 给出离屏、拉伸和恢复建议
- 导出 CSV

## 数据位置

`%APPDATA%\TenNativeApps\computer-injury-ledger\records.tsv`

## 使用方式

1. 双击 `01-ComputerInjuryLedger.exe`
2. 填写当天电脑工作和身体反馈
3. 点击“保存记录”
4. 查看右侧评分和建议
5. 需要整理时点击“导出 CSV”
