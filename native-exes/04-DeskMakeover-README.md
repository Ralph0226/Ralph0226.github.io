# 工位微装修

对应程序：`04-DeskMakeover.exe`

## 用途

帮助用户用低预算改善桌面环境、人体工学、光线和收纳，让工位更舒适、更高效。

## 适合人群

- 办公室白领
- 程序员、研究生、居家办公人群
- 肩颈、眼睛、腰背不适的人

## 核心功能

- 记录使用场景
- 记录显示器数量和改造预算
- 记录主要不适
- 记录光线和整洁评分
- 自动生成工位评分
- 给出改造优先级
- 导出 CSV

## 数据位置

`%APPDATA%\TenNativeApps\desk-makeover\records.tsv`

## 使用方式

1. 双击 `04-DeskMakeover.exe`
2. 填写当前工位情况和预算
3. 保存后查看工位评分
4. 根据建议优先改造显示器高度、光线和桌面动线
