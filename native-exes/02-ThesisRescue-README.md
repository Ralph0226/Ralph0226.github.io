# 论文拖延急救箱

对应程序：`02-ThesisRescue.exe`

## 用途

把论文写作压力拆成今天能完成的小任务，帮助本硕博用户推进选题、开题、实验、初稿、修改和答辩准备。

## 适合人群

- 本科毕业论文作者
- 硕士、博士论文作者
- 导师催稿、盲审、答辩前焦虑的人

## 核心功能

- 记录论文题目和学位阶段
- 记录当前论文阶段
- 记录截止日期和完成度
- 记录当前卡点
- 自动生成推进安全度
- 给出今日最小任务建议
- 导出 CSV

## 数据位置

`%APPDATA%\TenNativeApps\thesis-rescue\records.tsv`

## 使用方式

1. 双击 `02-ThesisRescue.exe`
2. 填写论文题目、阶段、截止日期和完成度
3. 写下当前卡点
4. 保存后查看今日推进建议
5. 每天更新一次进度
