# 跨屏剪贴管家

对应程序：`08-ClipManager.exe`

## 用途

集中保存工作、学习中零散的链接、文字、截图说明和灵感，避免资料散落后找不到。

## 适合人群

- 研究生、博士、写作者
- 产品经理、运营、设计师
- 经常复制链接、截图和资料的人

## 核心功能

- 记录资料标题
- 标记资料类型
- 添加标签
- 保存来源链接
- 保存正文或截图说明
- 自动生成整理度评分
- 导出 CSV

## 数据位置

`%APPDATA%\TenNativeApps\clip-manager\records.tsv`

## 使用方式

1. 双击 `08-ClipManager.exe`
2. 遇到有用资料时快速保存
3. 给资料添加标签
4. 定期搜索、整理和导出
