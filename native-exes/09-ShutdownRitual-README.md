# 睡前收工仪式

对应程序：`09-ShutdownRitual.exe`

## 用途

帮助白领在睡前完成工作收尾，把今日完成、未完成、担忧和明日三件事写清楚，降低睡前焦虑。

## 适合人群

- 下班后还想着工作的人
- 睡前容易焦虑的人
- 明天任务很多但脑子停不下来的人

## 核心功能

- 记录今日完成
- 记录未完成事项
- 记录明日三件事
- 记录担忧暂存箱
- 记录情绪和断屏时间
- 自动生成收工完成度
- 导出 CSV

## 数据位置

`%APPDATA%\TenNativeApps\shutdown-ritual\records.tsv`

## 使用方式

1. 双击 `09-ShutdownRitual.exe`
2. 睡前用 3 分钟填写当天收工内容
3. 保存后查看收工完成度
4. 明天打开程序继续执行三件事
