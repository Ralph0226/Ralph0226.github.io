# 刚需效率软件合集

这是根据 `ten-miniapp-app-prds.md` 开发的 10 个微信小程序/App 原型合集。项目是纯静态前端，可直接托管到 GitHub Pages，也可以继续拆成微信小程序页面。

## 包含的 10 个工具

1. 电脑工伤账本
2. 论文拖延急救箱
3. 打工人能量预算
4. 工位微装修
5. 会议废话压缩器
6. 简历战斗力雷达
7. 今日不背锅
8. 跨屏剪贴管家
9. 睡前收工仪式
10. 学习工作复利账本

## 功能

- 每个工具都有独立录入表单
- 本地自动保存，不需要服务器
- 即时诊断评分和建议
- 记录编辑、复制、删除
- JSON 数据导出
- 一键复制传播文案
- 适配电脑和手机浏览器

## 本地打开

直接打开：

`index.html`

## GitHub Pages

把本目录上传到 GitHub 仓库后，在仓库 Settings -> Pages 中选择：

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/root`

然后访问 GitHub Pages 链接即可。
